## Tasks

### Task 01-02-A: Refactor PetStage for multi-canvas architecture

<action>
重构 `PetStage.tsx`，从 `Live2DRenderer` 切换到 `RiveRenderer`。

核心变更：

1. **替换 import** — `Live2DRenderer` → `RiveRenderer`
2. **移除 `capability-resolver` 引用** — Rive SM 管理动画，不需要单独的 capability 解析
3. **canvas 容器** — `canvasContainerRef` 包含两个子 canvas（RiveRenderer 创建）
4. **渲染生命周期** — `useEffect` 调用 `renderer.loadModel()` → `renderer.resize()` → action playback
5. **模型切换** — `switchModel` 替换为调用 `renderer.loadModel() + resize() + playAction()`
6. **移除 `getModelCanvasSize` 引用** — Rive 模型尺寸由 runtime 管理
7. **Loading 提示** — Live2D 改为 Rive

关键代码修改点：

**Import 替换：**
```typescript
// 移除
import { Live2DRenderer } from '../features/pet/Live2DRenderer';
import { resolveExpressionCapability, resolvePropCapabilities } from '../features/pet/capability-resolver';
import { getModelCanvasSize, ModelConfig } from '../features/pet/model-registry';

// 替换为
import { RiveRenderer } from '../features/pet/RiveRenderer';
import { ModelConfig } from '../features/pet/model-registry';
```

**Renderer 初始化（models useEffect）：**
```typescript
useEffect(() => {
  if (models.length === 0) return;

  setModelLoaded(false);
  setLoadError(null);
  let cancelled = false;

  const initRenderer = async () => {
    const renderer = new RiveRenderer();
    rendererRef.current = renderer;

    const model = models[Math.min(modelIndex, models.length - 1)] ?? models[0];
    await renderer.loadModel(model);
    if (cancelled || rendererRef.current !== renderer) return;

    // 将 renderer 创建的 canvas 挂载到容器
    const container = canvasContainerRef.current;
    if (container && renderer.view) {
      container.innerHTML = ''; // 清空容器
      // 背景 canvas 在前，角色 canvas 在后（CSS 层叠）
      // 角色 canvas 带有 pointer-events
      container.appendChild(renderer.bgCanvas);
      container.appendChild(renderer.view);
      setModelLoaded(true);
    }
  };

  initRenderer().catch((err) => { /* 错误处理 */ });

  return () => {
    cancelled = true;
    rendererRef.current?.destroy();
    rendererRef.current = null;
  };
}, [models]);
```

**Action 播放（actionRevision useEffect）：**
```typescript
useEffect(() => {
  if (!rendererRef.current) return;
  rendererRef.current.playAction(currentAction);
}, [actionRevision, currentAction]);
```

**移除的 useEffect 块：**
- 整个 `currentExpression` + `expressionRevision` useEffect — Rive SM 管理
- 整个 `currentProps` + `propsRevision` useEffect — Phase 2 处理
- `getOpaqueBoundsClient()` 调用 — Rive 有不同 API

**模型切换（modelIndex useEffect）：**
```typescript
useEffect(() => {
  if (prevModelIndex.current === null || prevModelIndex.current === modelIndex) {
    prevModelIndex.current = modelIndex;
    return;
  }
  prevModelIndex.current = modelIndex;

  const nextModel = models[modelIndex];
  if (!nextModel || !rendererRef.current) return;

  let cancelled = false;
  const switchModel = async () => {
    const renderer = rendererRef.current!;
    setModelLoaded(false);
    renderer.destroy();

    const newRenderer = new RiveRenderer();
    rendererRef.current = newRenderer;
    await newRenderer.loadModel(nextModel);
    if (cancelled) return;

    const container = canvasContainerRef.current;
    if (container && newRenderer.view) {
      container.innerHTML = '';
      container.appendChild(newRenderer.bgCanvas);
      container.appendChild(newRenderer.view);
    }
    setModelLoaded(true);
  };
  switchModel().catch(console.error);

  return () => { cancelled = true; };
}, [modelIndex, models]);
```

**Resize（petScale useEffect）：**
```typescript
useEffect(() => {
  if (!rendererRef.current) return;
  const scale = parseFloat(document.documentElement.dataset.petScale || '1');
  const w = Math.round(520 * scale);     // 默认宽度
  const h = Math.round(760 * scale);     // 默认高度
  rendererRef.current.resize(w, h);
}, [modelLoaded]);
```

**Loading 提示文本：**
```typescript
{loadError ? `Rive load failed: ${loadError}` : 'Loading Rive...'}
```

**className 更新：**
```typescript
className="rive-container"  // 替代 live2d-container
```

### Task 01-02-B: Add placeholder .riv model

<action>
在 `public/models/` 下准备 Rive 模型结构：

1. 在 `public/assets/models/models.json` 中添加 Rive 模型条目（参考现有格式）
2. 放置一个占位 `.riv` 文件到 `public/models/placeholder/placeholder.riv`

由于用户自己提供 .riv 文件，这里需要创建一个最小可用占位。有两种方式：
- **方式 A（推荐）**: 使用 Rive 官方的示例 .riv 文件（从 Rive 编辑器或 rive-app GitHub 仓库获取）
- **方式 B**: 创建一个极简配置的模型注册表条目，等用户提供 .riv 文件后再填充

实现方式 B：
```json
// public/assets/models/models.json 添加条目
{
  "id": "placeholder",
  "name": "Rive Pet",
  "path": "vivipet-assets://models/placeholder/placeholder.riv",
  "type": "rive",
  "window": { "width": 520, "height": 760 }
}
```

在 `model-registry.ts` 中添加 Rive 类型支持：
```typescript
export type ModelType = 'live2d' | 'rive';

export interface ModelConfig {
  id: string;
  name: string;
  path: string;
  type?: ModelType;  // 可选，默认 'live2d'
  window?: { width: number; height: number };
}
```

设置 `type: 'rive'` 让加载器知道这个模型用 RiveRenderer。

注意：Task 01-02-A 中 `PetStage` 使用 `RiveRenderer` 创建实例，不再检查 model type。这意味着所有模型都用 RiveRenderer 加载。Phase 4 将处理类型分派。
</action>
