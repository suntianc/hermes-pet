    详细实现要求:

    **A. CubismModel 内部类（包装 CubismUserModel）:**
    - load(modelJsonUrl, canvasWidth, canvasHeight):
      * fetchArrayBuffer(url): fetch → arrayBuffer，参考旧实现模式
      * CubismModelSettingJson 解析 .model3.json
      * 加载 .moc3 → this.loadModel(mocBuffer)
      * 并行加载: 表情、物理、参数ID、motion
      * setupBreath(): 呼吸动画参数
      * setupLayout(): 模型布局
      * createRenderer(): WebGL renderer，设置 premultiplied alpha
      * loadShaders(SHADER_PATH): 加载 shader
      * waitForShaders(): 轮询等待 shader 就绪
      * loadTextures(): 加载模型纹理(GL.createTexture + texImage2D)
      * 记录 motionGroups 和 expressionNames

    - update(deltaTimeSeconds):
      * userTimeSeconds += deltaTime
      * _model.loadParameters()
      * 如果 motion 播放完成 → 自动回到 Idle
      * _motionManager.updateMotion → _expressionManager.updateMotion
      * _dragManager.update → 呼吸 → 物理 → _model.update

    - draw(canvasWidth, canvasHeight):
      * CubismMatrix44 projection 计算
      * setMvpMatrix → setRenderState → drawModel

    - startMotion(group, index, priority):
      * motions Map 查找 → _motionManager.startMotionPriority
      * 支持 PRIORITY_IDLE(1), PRIORITY_NORMAL(2), PRIORITY_FORCE(3)

    - hasMotionGroup(name): 检查 motionGroups 是否包含该组名

    动画循环:
    ```typescript
    private startAnimation(): void {
      this.lastFrameTime = performance.now();
      const animate = (time: number) => {
        if (this.disposed || !this.model || !this.gl || !this.canvas) return;
        const deltaSeconds = Math.min(0.064, Math.max(0.001, (time - this.lastFrameTime) / 1000));
        this.lastFrameTime = time;
        // WebGL 帧设置
        this.gl.viewport(0, 0, this.canvas.width, this.canvas.height);
        this.gl.clear(gl.COLOR_BUFFER_BIT | gl.DEPTH_BUFFER_BIT | gl.STENCIL_BUFFER_BIT);
        // 模型更新+绘制
        this.model.update(deltaSeconds);
        this.model.draw(this.canvas.width, this.canvas.height);
        this.animationFrame = requestAnimationFrame(animate);
      };
      this.animationFrame = requestAnimationFrame(animate);
    }
    ```
