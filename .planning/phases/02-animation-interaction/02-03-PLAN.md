<interfaces>
**CubismBreath API:**
```typescript
import { CubismBreath, BreathParameterData } from '@framework/effect/cubismbreath';

// 工厂方法
CubismBreath.create(): CubismBreath

// 配置呼吸参数（将哪些 Live2D 参数关联到呼吸效果）
// 一般用 ParamBodyAngleX、ParamBodyAngleY、ParamBreath 等
setParameters(breathParameters: BreathParameterData[]): void

// 每帧调用（传入 deltaTimeSeconds）
updateParameters(model: CubismModel, deltaTimeSeconds: number): void

// BreathParameterData 构造:
new BreathParameterData(
  parameterId: CubismIdHandle,  // 参数 ID
  offset: number,               // 正弦波偏移（默认值决定"
  peak: number,                 // 波峰高度（振幅强度）
  cycle: number,                // 周期（秒，完整正弦波的时间）
  weight: number                // 作用权重（0.0~1.0）
)
```

**CubismEyeBlink API:**
```typescript
import { CubismEyeBlink } from '@framework/effect/cubismeyeblink';

// 工厂方法（传入 modelSetting 自动读取 EyeBlinkParameterIds）
// modelSetting 读取的 ParameterIds 在 constructor 中自动填充
CubismEyeBlink.create(modelSetting: ICubismModelSetting): CubismEyeBlink

// 配置眨眼间隔（默认 4.0 秒）
setBlinkingInterval(intervalSeconds: number): void

// 配置眨眼动作时长（关闭、闭合、打开三个阶段）
setBlinkingSetting(closing: number, closed: number, opening: number): void

// 每帧调用（传入 deltaTimeSeconds）
updateParameters(model: CubismModel, deltaTimeSeconds: number): void
```

**CubismModel:**
```typescript
// CubismModel 的 addParameterValueById 由 CubismBreath/EyeBlink 内部调用
// 注意：CubismModel 的参数值是叠加的，所以 breath 和 action motions 不冲突
```

**Default parameter IDs for breath (参考 Cubism 官方示例):**
| Parameter | Range | Purpose |
|-----------|-------|---------|
| ParamBodyAngleX | -30~30 | 身体左右摆动（呼吸时身体微动） |
| ParamBreath | 0~1 | 呼吸专用参数（如果有） |
| ParamAngleX | -30~30 | 头部左右转动 |
</interfaces>

**Current render loop (from Plan 02-02):**
```
startLoop → loop:
  updateMouseFollow()
  updateLipSync()
  renderer.drawModel()
  requestAnimationFrame(loop)
```

**Note on deltaTime calculation:**
CubismBreath.updateParameters() 需要 deltaTimeSeconds（距上一帧的秒数）。
需要在 startLoop 中加入时间追踪以计算精确 deltaTime。
CubismEyeBlink.updateParameters() 也需要 deltaTimeSeconds。
使用 performance.now() 或 Date.now() 计算帧间隔。

    ```typescript
    /** 初始化 CubismBreath 呼吸效果和 CubismEyeBlink 眨眼效果 */
    private initIdleEffects(): void {
      if (!this.userModel || !this.modelSetting) return;

      // ---- 初始化呼吸 ----
      try {
        const breath = CubismBreath.create();
        const idManager = CubismFramework.getIdManager();

        // 典型呼吸参数绑定（参考 Cubism SDK 官方示例）
        // ParamBodyAngleX: 身体微微左右摆动；ParamBreath: 胸部起伏
        const breathParams: BreathParameterData[] = [];

        // ParamBodyAngleX — 身体水平微动
        const bodyAngleXId = idManager.getId(CubismDefaultParameterId.ParamBodyAngleX);
        breathParams.push(new BreathParameterData(bodyAngleXId, 0.0, 2.0, 6.0, 0.5));

        // ParamBreath — 如果有这个参数，用于胸部起伏
        // 部分模型使用 ParamBreath ID
        try {
          const breathId = idManager.getId('ParamBreath');
          breathParams.push(new BreathParameterData(breathId, 0.5, 0.5, 6.0, 1.0));
        } catch {
          // 模型没有 ParamBreath 参数，跳过
        }

        // 可选: 影响 ParamAngleY 做微微点头
        try {
          const angleYId = idManager.getId(CubismDefaultParameterId.ParamAngleY);
          breathParams.push(new BreathParameterData(angleYId, 0.0, 1.0, 6.0, 0.3));
        } catch {
          // 跳过
        }

        // 可选: 影响 ParamMouthOpenY 做微张
        try {
          const mouthId = idManager.getId(CubismDefaultParameterId.MouthOpenY);
          breathParams.push(new BreathParameterData(mouthId, 0.0, 0.2, 8.0, 0.3));
        } catch {
          // 跳过
        }

        breath.setParameters(breathParams);
        this.breath = breath;
      } catch (err) {
        console.warn('[Live2DRenderer] Failed to init breath:', err);
        this.breath = null;
      }

      // ---- 初始化眨眼 ----
      try {
        // CubismEyeBlink.create(modelSetting) 会自动从 model3.json 读取 EyeBlinkParameterIds
        // 这会捕获 ParamEyeLOpen, ParamEyeROpen 等参数
        this.eyeBlink = CubismEyeBlink.create(this.modelSetting);
        // 设置眨眼间隔约 4 秒（默认值），动作时长: 闭合 100ms, 闭合停留 50ms, 打开 150ms
        this.eyeBlink.setBlinkingInterval(4.0);
        this.eyeBlink.setBlinkingSetting(0.1, 0.05, 0.15);
      } catch (err) {
        console.warn('[Live2DRenderer] Failed to init eyeBlink:', err);
        this.eyeBlink = null;
      }
    }
    ```

    ```typescript
    /** 每帧调用: 更新呼吸和眨眼参数 */
    private updateIdleEffects(deltaTimeSec: number): void {
      if (!this.userModel) return;
      const model = this.userModel.getModel();
      if (this.breath) {
        this.breath.updateParameters(model, deltaTimeSec);
      }
      if (this.eyeBlink) {
        this.eyeBlink.updateParameters(model, deltaTimeSec);
      }
    }
    ```

    6. **重写 `startLoop()` 方法**，加入 deltaTime 追踪和 `updateIdleEffects` 调用：
    ```typescript
    private startLoop(): void {
      this.lastFrameTime = performance.now();
      const loop = (now: number) => {
        if (this.disposed) return;
        if (this.userModel) {
          // 计算帧间隔（秒），限制最大值为 100ms 防止切换标签页后跳变
          const deltaTimeSec = Math.min((now - this.lastFrameTime) / 1000, 0.1);
          this.lastFrameTime = now;

          // 按依赖顺序更新
          this.updateMouseFollow();
          this.updateLipSync();
          this.updateIdleEffects(deltaTimeSec);
          this.userModel.getRenderer().drawModel();
        }
        this.animationId = requestAnimationFrame(loop);
      };
      this.animationId = requestAnimationFrame(loop);
    }
    ```
